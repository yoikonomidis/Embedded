
Ethernet Board Support Definitions
**********************************

scope:
   General Use

description:
   Ethernet configuration defines for XMOS development boards

keywords:
   ethernet, mii, mac

This module contains a header file that defines port initializers and
settings to use with XMOS ethernet components for common XMOS
development boards. The supported boards in this module are:

   * sliceKIT Core Board (L2) + Ethernet Slice

   * XR-AVB-LC-BRD (AVB Low Cost Development Board)

   * XC-2 Development Board
